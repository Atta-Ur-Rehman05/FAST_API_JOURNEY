## 🚀 Getting Started

1. Create a virtual environment

```bash
python -m venv .venv
```

2. Activate the environment

```bash
.venv\Scripts\activate
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
uvicorn main:app --reload
```
